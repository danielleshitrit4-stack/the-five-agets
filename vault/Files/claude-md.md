# CLAUDE.md

## תפקיד

מסמך ההנחיה המרכזי של הפרויקט עבור Claude Code. כל instance של Claude Code שנפתח בתיקייה הזו טוען את הקובץ הזה אוטומטית כחלק מהקונטקסט שלו.

## מה אמור להיות בו

- **תיאור הפרויקט** — מה הוא עושה, למי הוא מיועד
- **Tech stack** — שפות, פריימוורקים, ספריות מרכזיות
- **פקודות נפוצות** — build, test, lint, dev server
- **ארכיטקטורה** — תמונה גדולה של איך הקוד מאורגן (לא דברים שאפשר לגלות לבד)
- **קונבנציות** — דברים לא-טריוויאליים בארגון הקוד
- **אינטגרציות חיצוניות** — APIs, שירותים, MCP servers

## מצב נוכחי

הקובץ מגדיר את **פרסונת ראובן** — מנכ"ל וירטואלי של מערך הסוכנים של Zerem Academy. ~13 סקציות בעברית: זהות, טון, מסגרת החלטה (9 צעדים), בחירת sub-agent, מרשם (4 שורות [planned]), 5 מקרי קצה, פרוטוקול vault עם הכרעת "vault wins", checklist בקרת איכות, פלט מובנה, anti-patterns, הקשר עסקי, וקבצים קשורים. הוקם ב-[[reuven-persona-bootstrap]].

## מי הבעלים

`project` — קובץ ברמת הפרויקט שנדחף ל-git ומשותף עם כל מי שעובד ברפו.
