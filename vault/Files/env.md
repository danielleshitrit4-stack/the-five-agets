# .env

## תפקיד

קובץ עם משתני סביבה לוקליים. כאן נכנסים API keys, tokens, וכל ערך רגיש שהקוד צריך בזמן ריצה.

## פורמט

```
KEY=VALUE
```

שורה לכל משתנה, ללא רווחים מסביב ל-`=`.

> [!warning] לא לדחוף ל-git
> הקובץ הזה **gitignored** ב-[[gitignore]]. הוא נשאר רק על המחשב שלך.

## מצב נוכחי

ריק — רק הערות הסבר. עוד לא הוגדרו משתנים בפועל.

## דוגמאות אופייניות

- `ANTHROPIC_API_KEY=sk-ant-...` — מפתח ל-Claude API
- `OPENAI_API_KEY=sk-...` — מפתח ל-OpenAI
- כל שירות חיצוני שדורש הזדהות

## מי הבעלים

`project` — נחוץ לפרויקט, אבל **אישי** לכל מפתח. כל מפתח יוצר עותק משלו עם המפתחות שלו.

## קבצים קשורים

- [[gitignore]] — מסתיר את הקובץ הזה מ-git
- [[claude-md]] — אמור (בעתיד) לתעד אילו משתנים הפרויקט דורש
