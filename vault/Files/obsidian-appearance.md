# .obsidian/appearance.json

## תפקיד

הגדרות **עיצוב ויזואלי** של Obsidian: theme (light/dark), CSS snippets פעילים, פונט, גודל טקסט.

## מצב נוכחי

```json
{}
```

ריק — Obsidian משתמש בברירות המחדל. ברגע שתעבור theme או תפעיל CSS snippet, הקובץ ייכתב מחדש.

## מה הקובץ יכיל בעתיד

- `theme: "obsidian"` או `"moonstone"` — מצב כהה/בהיר
- `cssTheme: "Minimal"` — שם של community theme
- `enabledCssSnippets: ["snippet-name"]` — CSS snippets פעילים
- `baseFontSize: 16` — גודל טקסט בסיס
- `interfaceFontFamily: "..."` — פונט הממשק
- `textFontFamily: "..."` — פונט הטקסט בתוך פתקים

## מי הבעלים

`obsidian` — נוצר ומנוהל ע"י Obsidian (Settings → Appearance).
