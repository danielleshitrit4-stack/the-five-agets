# .obsidian/core-plugins.json

## תפקיד

מצב הפעלה (true/false) של כל ה-**Core Plugins** של Obsidian — הפיצ'רים המובנים שבאים עם האפליקציה.

## Plugins דלוקים בפרויקט

| Plugin | מטרה |
|---|---|
| `file-explorer` | סייר הקבצים מצד שמאל |
| `global-search` | חיפוש כללי בכל ה-vault |
| `switcher` | Quick Switcher (Ctrl+O) |
| `graph` | תצוגת הגרף של הקישורים |
| `backlink` | הצגת backlinks לכל פתק |
| `canvas` | קנבסים אינסופיים לתכנון ויזואלי |
| `outgoing-link` | הצגת קישורים יוצאים |
| `tag-pane` | חלונית התגיות |
| `properties` | תצוגת frontmatter properties |
| `page-preview` | תצוגה מקדימה ב-hover |
| `daily-notes` | פתקים יומיים מתוארכים |
| `templates` | תבניות לפתקים חדשים |
| `note-composer` | פיצול/מיזוג של פתקים |
| `command-palette` | פלטת פקודות (Ctrl+P) |
| `editor-status` | סטטוס בר של העורך |
| `bookmarks` | סימניות לפתקים |
| `outline` | תצוגת התוכן (headings) של הפתק הנוכחי |
| `word-count` | ספירת מילים |
| `file-recovery` | שחזור גרסאות |
| `sync` | Obsidian Sync (סנכרון בענן) |
| **`bases`** | **תצוגות database-like** (חדש יחסית — מרכזי ל-[[skill-obsidian-bases]]) |

## Plugins כבויים

`footnotes`, `slash-command`, `markdown-importer`, `zk-prefixer`, `random-note`, `slides`, `audio-recorder`, `workspaces`, `publish`, `webviewer` — לא בשימוש בפרויקט הזה.

## מי הבעלים

`obsidian` — Obsidian כותב את הקובץ הזה כל פעם שמדליקים/מכבים plugin ב-Settings → Core plugins.
